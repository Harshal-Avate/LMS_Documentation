My Courses

The "My Courses" section is your central hub for academic progression within LearnOS. Here, you can track your current enrollment status, monitor individual module progress, and access detailed curriculum resources for each of your active learning paths.

info

Important Update

Course progression is now calculated in real-time based on both attendance and assessment completion. Ensure you synchronize your offline project commits to see the latest data.

Each course in the LearnOS ecosystem is divided into specific tracks. Your dashboard provides a high-level visualization of where you stand in each track. The following table summarizes the primary data points available in your current semester:

API Integration for Data Sync

If you are building custom tooling or using the LearnOS CLI, you can programmatically fetch your course status using our secure API endpoint. Use the following payload structure for authentication:

Request Payload - JSON 
{
  "endpoint": "/api/v1/student/courses",
  "method": "GET",
  "headers": {
    "Authorization": "Bearer <TOKEN>",
    "X-LearnOS-Version": "2.4.0"
  },
  "params": {
    "include_analytics": true,
    "filter": "active"
  }
}

Tracking Course Modules

Under each course header, you will find a list of modules. These are the building blocks of your curriculum. Clicking into a module will reveal the required reading, associated labs, and assessment deadlines. Remember that some modules require prerequisites to be completed before unlocking.
