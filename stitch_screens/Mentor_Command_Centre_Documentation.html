Mentor Command Centre (Dashboard)

The Mentor Command Centre serves as your central hub for managing student engagement, course progress, and administrative tasks. It provides a real-time overview of your educational ecosystem.

info

This dashboard is only accessible to users with the Mentor role assigned by an Admin.

Dashboard overview

The interface is divided into functional zones that allow for rapid context switching. The top bar contains global search and notification controls, while the primary workspace is dominated by data visualization and student queues.

Key metrics panel

Metrics are tracked asynchronously to ensure UI responsiveness. Key performance indicators are aggregated at the mentor level.
