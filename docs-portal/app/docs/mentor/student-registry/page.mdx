# Student Registry & Management

The Student Registry provides mentors with a complete list of their assigned students, including contact information, enrollment details, and quick access to student profiles.

## How to Access

From the sidebar under the **CORE** group, click **My Students** or the equivalent student list option.

## Interface Overview

| Element | Description |
|---------|-------------|
| **Student Registry Table** | Searchable list of all assigned students |
| **Search & Filter Bar** | Filter students by name, email, or course |
| **Status Indicators** | Active, Completed, or Dropped badges |
| **Student Profile Drawer** | Slide-out panel with detailed student information |
| **Chat Button** | Direct link to open a chat conversation |

## Student Table Columns

| Column | Description |
|--------|-------------|
| **Name** | Student's full name |
| **Email** | Registered email address |
| **Phone** | Contact number |
| **Course** | Enrolled course name |
| **Enrollment Date** | When the student joined |
| **Status** | Active / Completed / Dropped |
| **Actions** | View Profile, Chat buttons |

## Viewing a Student Profile

**Step 1** — Locate the student in the registry table using the search input.

**Step 2** — Click on the student's name or the **View Profile** button.

**Step 3** — The student profile slide-out panel displays:
  - Personal details (name, email, phone)
  - Course enrollment and progress
  - Attendance percentage
  - Recent assessment submissions
  - Certificate status

**Step 4** — Click **Open Chat** to start a direct conversation with the student.

## Filtering Students

| Filter | Description |
|--------|-------------|
| **Search by Name/Email** | Type in the search bar to filter the list |
| **Status Filter** | Show only Active, Completed, or Dropped students |
| **Course Filter** | Show only students enrolled in a specific course |

<Info title="Assigned Students Only">
The Student Registry shows only students **assigned to you**. You cannot view or contact students assigned to other mentors unless authorized by an admin.
</Info>
