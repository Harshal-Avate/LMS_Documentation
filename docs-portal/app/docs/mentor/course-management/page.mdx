# Course Management Hub

![Visual Guide](/images/mentor/course-management-hub.png)

The Course Hub displays all academic courses assigned to the logged-in mentor. It acts as the gateway to syllabus editing, course analytics, and enrolled student directories.

## How to Access

From the sidebar under the **ACADEMICS** group, click **My Courses** (path `/mentor/my-courses`).

## Interface Overview

| Element | Description |
|---------|-------------|
| **Grid Layout** | Displays course cards for all assigned courses |
| **Course Card** | Header thumbnail, title/description body, "Manage Course" button footer |
| **Empty State** | Dotted container indicating "No courses assigned" if empty |

## Course Card Details

Each course card shows:

| Element | Description |
|---------|-------------|
| **Thumbnail** | Course cover image |
| **Category** | The course category or technology area |
| **Title** | Course name |
| **Description** | Brief course overview |
| **Manage Course Button** | Opens the Course Detail/Profile page |

## Available Operations

### Browse Assigned Courses

Scroll through the course card grid to see all courses assigned to you by the administrator.

### Access Course Management Portal

**Step 1** " Browse through the available course catalog cards on the grid.

**Step 2** " Hover over any card to trigger a smooth scaling animation.

**Step 3** " Click the **card** or the **Manage Course** button to drill down into:
  - Course Profile & Enrollee Analytics
  - Syllabus Curriculum Builder
  - Enrolled Student List

## Course Status Indicators

Courses may display status badges:

| Status | Description |
|--------|-------------|
| **Draft** | Course is in creation mode, not visible to students |
| **Pending Approval** | Submitted for admin review |
| **Rejected** | Admin has rejected the submission with feedback |
| **Published** | Live and visible to enrolled students |

<Note title="Read-Only Listing">
Courses are **read-only** on the Course Hub grid. Allocation maps (which courses are assigned to which mentor) are configured by Administrators. Contact your admin if you need a new course assigned.
</Note>

<Info title="Course Count">
The Course Hub shows a count badge indicating how many courses are currently assigned to you. If this is 0, contact your administrator.
</Info>
