# Portal Announcements Board

![Visual Guide](/images/mentor/portal-announcements-board.png)

Create and manage announcements for the mentor portal. This section provides a complete step-by-step guide to using this feature in the SimpleSphere Mentor Portal.

## How to Access

Navigate to this section from the left sidebar in the Mentor Portal. Refer to the sidebar navigation under the appropriate group.

## Overview

This feature provides mentors with dedicated tools to manage their responsibilities efficiently. Below is a summary of the key capabilities:

| Feature | Description |
|---------|-------------|
| **Primary Function** | Create and manage announcements for the mentor portal. |
| **Access Level** | Mentor and above |
| **Location** | Left sidebar navigation |

## Key Steps

**Step 1** " Navigate to this section from the mentor sidebar.

**Step 2** " Review the current data and status indicators on the page.

**Step 3** " Use the available action buttons to perform tasks.

**Step 4** " Save or submit your changes as prompted.

## Important Notes

<Info title="Mentor Access">
This feature is available exclusively to **Mentor** and **Admin** accounts. Student accounts do not have access to this section.
</Info>

<Note title="Data Sync">
Changes made in this section sync with the database in real time. Refresh the page if data appears outdated.
</Note>

For detailed documentation of this feature, please refer to the **SimpleSphere Mentor Guidebook Version 1.0 (May 2026)**.
