# Dashboard Overview

After logging in, you will be automatically redirected to your role-specific dashboard. Each portal has a unique dashboard designed for its users.

## Dashboard by Role

### Student Dashboard

The student home page provides a personalized summary of your learning activity.

**Key sections:**

| Section | Description |
|---------|-------------|
| **Live Class Banner** | Appears when a mentor has an active Jitsi session |
| **Survey Notification** | Prompts you to complete pending feedback surveys |
| **Stat Cards** | Enrolled Courses, Attendance %, Pending Tasks, Certificates |
| **Continue Learning** | List of enrolled courses with resume buttons |
| **Upcoming Sessions** | Scheduled live classes with join links |
| **Pending Assessments** | Overdue tasks highlighted in red |
| **Quick Actions** | Shortcuts to AI Mentor, Analytics, Code Arena |

### Mentor Dashboard (Command Centre)

The Mentor Dashboard is a comprehensive operations hub.

**Key sections:**

| Section | Description |
|---------|-------------|
| **System Status Badge** | Shows "SYSTEM OPERATIONAL" status |
| **Global Actions** | Quick Meeting, Schedule, Reports buttons |
| **Analytical Cards** | Assigned Students, Active Courses, Pending Assignments, Messages |
| **SLA Response Monitor** | Tracks feedback latency and grading queue |
| **Charts** | Student Engagement area chart, Tasks Status pie chart |
| **AI Assistant Card** | Shortcut to launch AI Co-Pilot |
| **Recent Activity Feed** | Timeline of recent student activity |
| **Schedule Manager** | Upcoming Jitsi sessions with Edit/Cancel/Join |

### Admin Dashboard (Control Deck)

The Admin Dashboard displays system-wide operational metrics.

**Key sections:**

| Section | Description |
|---------|-------------|
| **Center Switcher** | Filter data by educational center |
| **Overview Stat Cards** | Total Students, Mentors, Courses, Revenue |
| **Enrollment Trends Graph** | Interactive chart of student sign-ups over time |
| **Center Performance Leaderboard** | Quality scores by center |
| **Recent Leads List** | Newly registered users |
| **System Activity Feed** | Log of recent administrative changes |

## First Steps After Login

Follow these recommended steps when accessing SimpleSphere for the first time:

**Step 1** — Review your dashboard stat cards to understand your current status.

**Step 2** — Check for any announcements or notifications at the top of your dashboard.

**Step 3** — Navigate to your primary section (Courses for Students, My Courses for Mentors, Users for Admins).

**Step 4** — Update your profile and password in **Settings / Preferences** if this is your first login.

<Note title="Default Theme">
SimpleSphere defaults to your system's preferred color scheme (light or dark). You can manually override this using the theme toggle in the top-right corner of any page.
</Note>
