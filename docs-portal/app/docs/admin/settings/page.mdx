# Settings & Preferences

![Visual Guide](/images/admin/settings-and-preferences.png)

The Settings module is the personal account console for administrative staff to update security credentials and review account clearance.

## How to Access

- Select **'Preferences'** under the **SYSTEM** group in the left sidebar.

## UI Elements

| Element | Description |
|---------|-------------|
| **Tab Navigators** | Switch between **Student Identity** and **Security Protocols** tabs |
| **Identity Card** | Displays your official name, username, and clearance level |
| **Security Form** | Password change fields |
| **Initialize Button** | Saves the updated security password |

## Viewing Your Identity

The **Student Identity** tab (or Admin Identity) displays:

| Field | Description |
|-------|-------------|
| **Full Name** | Your official registered name |
| **Username** | Your login username |
| **Level of Clearance** | Your admin access tier (Super Admin / Center Admin) |
| **Assigned Centers** | Educational centers you have administrative access to |

## Changing Your Password

**Step 1** " Select the **'Security Protocols'** tab.

**Step 2** " Input your **current password** in the **'Current Access Key'** field.

**Step 3** " Enter your **new desired password** in the **'New Access Key'** field.

**Step 4** " Confirm the password by typing it again in the **'Confirm New Key'** field.

**Step 5** " Click **'Initialize Protocol Update'** to apply the change.

## Password Strength Requirements

| Requirement | Description |
|-------------|-------------|
| **Minimum Length** | At least 8 characters |
| **Uppercase** | At least one uppercase letter |
| **Numbers** | At least one numeric digit |
| **Special Characters** | At least one special character (@, #, !, $, etc.) |

<Warning title="Admin Security is Critical">
Choose a **strong, unique password** for your admin account. Admin credentials provide access to sensitive student data, financial records, and system configurations. A compromised admin account can cause serious data breaches.
</Warning>

## After Password Change

- The new password takes effect **immediately**
- You will be prompted to log in again with your new credentials
- Other active sessions may be invalidated depending on configuration

<Info title="Two-Factor Authentication">
If your institution requires 2FA (Two-Factor Authentication) for admin accounts, configure it in the Security Protocols tab. Contact your Super Admin if you need 2FA enabled.
</Info>
