# Assessments Management

![Visual Guide](/images/admin/assessments-management.png)

The Assessments Management module is a Kanban board interface that allows admins to assign tasks to mentors and track deliverable progress.

## How to Access

- Select **'Tasks'** under the **CORE** group in the left sidebar.

## UI Elements

| Element | Description |
|---------|-------------|
| **Assign Task Button** | Opens the task creation form modal |
| **Kanban Columns** | Four status columns: TO DO, IN PROGRESS, REVIEW, DONE |
| **Task Cards** | Visual task representations with priority badges |
| **Task Form Modal** | Inputs for Title, Description, Assigned Mentor, and Priority |

## Kanban Board Columns

| Column | Description |
|--------|-------------|
| **TO DO** | Tasks created but not yet started by the mentor |
| **IN PROGRESS** | Tasks the mentor is actively working on |
| **REVIEW** | Tasks submitted by the mentor for admin review |
| **DONE** | Completed and approved tasks |

## Creating a New Task

**Step 1** " Click the **'Assign Task'** button at the top of the Kanban board.

**Step 2** " Input the **Task Title** and compile detailed instructions in the **Description** field.

**Step 3** " Select the target **Mentor** from the dropdown list.

**Step 4** " Choose a **priority level**:
  - 🔴 **High** " urgent tasks requiring immediate attention
  - 🟡 **Medium** " standard priority
  - ⚪ **Low** " non-urgent tasks

**Step 5** " Click **'Create Task'** to add the card to the **TO DO** column.

## Moving Tasks Across the Board

Tasks move through the Kanban pipeline using drag and drop:

**Step 1** " Click and hold a task card.

**Step 2** " Drag it to the target column (e.g., from TO DO to IN PROGRESS).

**Step 3** " Release the card to drop it in the new column.

**Step 4** " The status updates instantly and the mentor is notified.

<Note title="Mentor Updates">
Mentors can also drag their own task cards from TO DO → IN PROGRESS → DONE as they complete work. Admins can monitor the board in real time.
</Note>

## Task Card Information

Each task card displays:

| Field | Description |
|-------|-------------|
| **Title** | The name of the task |
| **Description** | Instructions and requirements |
| **Priority Badge** | High / Medium / Low color indicator |
| **Assigned Mentor** | Who the task is assigned to |
| **Created Date** | When the task was created |

<Info title="Drag and Drop">
Drag and drop task cards across the Kanban board columns to update their progress state in real time. Changes sync immediately with the backend.
</Info>
