# Course Management

![Visual Guide](/images/admin/course-management.png)

The Course Management panel is the catalog tool to create, edit, structure, and assign mentors to courses on the SimpleSphere platform.

## How to Access

- Select **'Course Hub'** under the **ACADEMICS** group in the left sidebar.

## UI Elements

| Element | Description |
|---------|-------------|
| **Course Table** | Grid with thumbnails, levels, prices, visibility, and enrollment counts |
| **Create Course Button** | Action button to add a new course |
| **Search & Filter Panel** | Input boxes to filter by category and status |
| **Course Form Modal** | Full form for course creation and editing |

## Course Table Columns

| Column | Description |
|--------|-------------|
| **Thumbnail** | Course cover image |
| **Title** | Course name |
| **Level** | Beginner / Intermediate / Advanced |
| **Price** | Course price (or FREE) |
| **Visibility** | Draft / Published |
| **Enrolled** | Number of students currently enrolled |
| **Actions** | Edit and Delete icons |

## Creating a New Course

**Step 1** " Click the **'Create Course'** button at the top of the page.

**Step 2** " Fill in the **Course Title** and **Description**.

**Step 3** " Select a **Category** from the dropdown.

**Step 4** " Select the course **Level**:
- Beginner
- Intermediate
- Advanced

**Step 5** " Enter the **Duration** (e.g., "3 Months", "12 Weeks").

**Step 6** " Toggle the **'Is Paid'** switch and enter the **Price** if applicable.

**Step 7** " Select the assigned **Mentor(s)** from the dropdown.

**Step 8** " Set **Visibility** to Draft (hidden) or Published (visible to students).

**Step 9** " Click **'Save'** to register the course.

<Note title="Draft vs Published">
Only courses marked as **"Published"** are visible in the student portal. Keep courses as **Draft** during content creation to prevent students from seeing incomplete material.
</Note>

## Editing an Existing Course

**Step 1** " Click the **Edit (pencil)** icon on the course row.

**Step 2** " Modify any fields in the form modal.

**Step 3** " Click **'Save'** to apply changes.

## Setting Prerequisites

In the Course Form Modal, you can set **prerequisites** " other courses that students must complete before enrolling in this course. Select prerequisite courses from the multi-select dropdown.

## Learning Outcomes

Add **Learning Outcomes** to describe what students will achieve by completing the course. These are displayed on the course card in the student portal.

<Info title="Mentor Assignment">
Courses must have at least one mentor assigned before they can be published. Mentors can view and manage courses assigned to them from their Mentor Portal.
</Info>
