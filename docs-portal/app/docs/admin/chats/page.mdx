# Chats / Messaging

![Visual Guide](/images/admin/chats-messaging.png)

The Chats module allows administrators to quickly locate student contact information and initiate direct communication via WhatsApp.

## How to Access

- Select **'Students'** under the **CORE** group in the left sidebar.

## UI Elements

| Element | Description |
|---------|-------------|
| **Student Directory** | List of all students with query filters and status indicators |
| **Student Profile Card** | Slide-out panel with course status, contact details, and registration dates |
| **WhatsApp Icon Button** | Direct contact link that initiates a WhatsApp chat |

## Locating and Contacting a Student

**Step 1** " Navigate to the **Students** page via the CORE menu.

**Step 2** " Use the **search bar** to locate the student by name, email, or phone.

**Step 3** " Click on the student's **name or row** to slide open their detail panel.

**Step 4** " The slide-out profile card displays:
  - Student name and photo
  - Course enrollment status
  - Contact details (phone, email)
  - Registration date
  - Payment status

**Step 5** " Click the **WhatsApp button** to open a direct messaging channel in WhatsApp.

## Student Profile Card Details

| Field | Description |
|-------|-------------|
| **Name** | Student's full name |
| **Phone** | Registered mobile number |
| **Email** | Email address |
| **Course** | Enrolled course name |
| **Status** | Active / Completed / Dropped |
| **Registration Date** | When the student was enrolled |
| **Payment Info** | Total fees, paid amount, balance |

<Note title="WhatsApp Integration">
The WhatsApp button auto-populates the student's phone number in WhatsApp Web or the WhatsApp app. Ensure the student's **country prefix** is correctly set (e.g., `+91` for India) to avoid messaging failures.
</Note>

## Filtering Students

Use these filters to narrow down the student list:

| Filter | Description |
|--------|-------------|
| **Search by Name** | Type the student's name in the search input |
| **Status Filter** | Filter by Active, Completed, or Dropped |
| **Center Filter** | Filter by educational center |
| **Course Filter** | Filter by enrolled course |

<Info title="Admin Messaging Note">
The admin chat module is primarily used for **contact lookup and outreach**. For bulk messaging or formal communications, use the **Announcements** module instead.
</Info>
