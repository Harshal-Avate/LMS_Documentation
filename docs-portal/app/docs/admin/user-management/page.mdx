# User Management

![Visual Guide](/images/admin/user-management.png)

The User Management workspace allows administrators to manage platform personnel, access privileges, and educational centers.

## How to Access

- **Students**: Select **'Students'** under the **CORE** group in the left sidebar.
- **Mentors**: Select **'Mentor Hub'** under the **CORE** group.
- **Roles/Access**: Select **'Access Control'** under the **SYSTEM** group.

## UI Elements

| Element | Description |
|---------|-------------|
| **Student/Mentor Registry Table** | Searchable lists with names, contact info, and status tags |
| **Create/Add User Button** | Opens a popup form to register new accounts |
| **Edit Action Icon** | Pencil icon to modify email, name, center, and role |
| **Deactivate/Delete Icon** | Trash/toggle to revoke access or change status |
| **Upload Vault** | Document section for verifying student IDs and credentials |

## Status Tags

| Status | Description |
|--------|-------------|
| **Active** | User has full access to their portal |
| **Completed** | Student has completed their course(s) |
| **Dropped** | User's access has been deactivated |

## Adding a New Student

**Step 1** " Navigate to the **Students** page under the CORE menu group.

**Step 2** " Click the **'Add Student'** button to open the registry form.

**Step 3** " Enter the following details:
- **Name** " Student's full name
- **Phone** " 10-digit mobile number (strictly validated)
- **Email** " Student's email address
- **Center** " Assign to the correct educational center
- **Total Fees** " Total tuition amount
- **Paid Amount** " Amount already paid
- **Balance** " Automatically calculated

**Step 4** " Click **'Save'** to create and activate the account.

<Note title="Validation Rules">
- Student **phone numbers must contain exactly 10 digits**
- **Fees cannot be negative** values
- Email addresses must be in valid format
</Note>

## Adding a New Mentor

**Step 1** " Navigate to **Mentor Hub** under the CORE group.

**Step 2** " Click **'Add Mentor'** to open the registration form.

**Step 3** " Fill in Name, Email, Phone, and assigned Center.

**Step 4** " Click **'Save'** to create the mentor account.

## Deactivating a User

**Step 1** " Search for the user in the registry table using the search input.

**Step 2** " Click the **Edit (pencil)** icon on their row.

**Step 3** " Toggle the status to **'Dropped'** or click the **Deactivate** button.

**Step 4** " The user's access is revoked immediately.

## Access Control (Roles)

Navigate to **'Access Control'** under SYSTEM to:
- View all role assignments
- Assign or revoke admin-level access
- Manage center-specific permissions

<Warning title="Irreversible Deactivation">
Deactivating a user revokes their access immediately. Students who are deactivated cannot access their courses or data until reactivated by an admin.
</Warning>
