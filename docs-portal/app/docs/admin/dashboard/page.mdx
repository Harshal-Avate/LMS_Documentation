# Admin Dashboard

![Visual Guide](/images/admin/admin-dashboard.png)

The Admin Dashboard is the central control deck, displaying operational highlights and system-wide metrics in real time.

## How to Access

- After logging in as Admin, you will land here automatically.
- Alternatively, click **'Overview'** under the **CORE** group in the left sidebar.

## Dashboard UI Elements

| Element | Description |
|---------|-------------|
| **Center Switcher Dropdown** | Filter all dashboard data by educational center |
| **Overview Stat Cards** | Key metrics: Total Students, Mentors, Courses, Collections |
| **Enrollment Trends Graph** | Interactive charts mapping student sign-ups over time |
| **Center Performance Leaderboard** | Student counts and quality scores by center |
| **Recent Leads List** | Table showing newly registered leads |
| **System Activity Feed** | Log of recent administrative changes |

## Overview Stat Cards

| Card | Description |
|------|-------------|
| **Total Students** | Count of all active enrolled students |
| **Total Mentors** | Count of all active mentor accounts |
| **Total Courses** | Count of all published courses |
| **Collections (Revenue)** | Aggregated fee collection across centers |

## Using the Center Switcher

The Center Switcher allows admins to filter all data by educational center:

**Step 1** " Locate the **Center Switcher** dropdown in the top header bar.

**Step 2** " Click the dropdown to see a list of all authorized educational centers.

**Step 3** " Select a center to dynamically filter all dashboard statistics and metrics.

**Step 4** " To return to the all-centers view, select **'All Centers'**.

<Note title="Super Admin vs Center Admin">
**Super Admins** can toggle and view data across all centers.
**Center Admins** are restricted to data from their specific assigned center only.
</Note>

## Enrollment Trends Graph

The interactive enrollment graph shows:
- Student sign-up trends over customizable time periods
- Comparison between centers (Super Admin view)
- Peaks and troughs in enrollment activity

## Center Performance Leaderboard

A ranking table showing:

| Column | Description |
|--------|-------------|
| **Center Name** | Name of the educational center |
| **Student Count** | Total enrolled students |
| **Quality Score** | Composite score based on SLA compliance and student performance |

## System Activity Feed

A real-time log of recent administrative actions including:
- New user registrations
- Course status changes
- Certificate issuances
- System configuration updates
