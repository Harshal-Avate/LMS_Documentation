# Code Arena

![Visual Guide](/images/students/code-arena.png)

The Code Arena is an in-browser coding environment where you can write, run, and test code directly inside the SimpleSphere platform " no local setup required.

## How to Access

- Click **'Code Arena'** in the left sidebar under **LEARNING RESOURCES**.

## Features

### Code Editor

A fully featured code editor with:
- **Syntax highlighting** for multiple programming languages
- **Line numbers** for easy reference
- **Auto-indentation** and code formatting
- **Dark/Light theme** synchronized with the platform theme

### Language Selection

Switch between supported programming languages using the language dropdown:

| Language | Supported |
|----------|-----------|
| Python | ✅ Yes |
| JavaScript | ✅ Yes |
| Java | ✅ Yes |
| C | ✅ Yes |
| C++ | ✅ Yes |
| TypeScript | ✅ Yes |

### Run Code Button

Click the **Run Code** button to execute your code against the built-in interpreter.

### Output / Console Panel

The output panel displays:
- **Standard output** " results of `print()` / `console.log()` statements
- **Error messages** " syntax or runtime errors in red
- **Return values** " values returned by your functions

## How to Use the Code Arena

**Step 1** " Navigate to **Code Arena** in the sidebar.

**Step 2** " Select your programming language from the language dropdown.

**Step 3** " Write your code in the editor panel.

**Step 4** " Click **Run Code** to execute.

**Step 5** " View the output in the console panel below the editor.

**Step 6** " Debug and iterate until your code produces the expected result.

## Coding Problems

Your mentor may assign specific coding problems through the Code Arena. These appear as structured challenges with:

- **Problem Statement** " description of what the code should do
- **Input Format** " expected input structure
- **Constraints** " limitations on values and size
- **Examples** " sample inputs and expected outputs
- **Test Cases** " automated tests that verify your solution

<Info title="Interactive Lab Integration">
The Code Arena is also accessible in **Split View / Interactive Lab** mode, where you can watch a course lecture video and write code side-by-side. Access this from the Dashboard or My Courses page.
</Info>

<Note title="Code is not saved automatically">
Your code in the Code Arena editor is **not automatically saved**. Copy your solutions to a local file if you want to keep them before closing the browser.
</Note>
