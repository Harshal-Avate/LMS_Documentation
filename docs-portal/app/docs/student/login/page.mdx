# Student Login

![Visual Guide](/images/students/login.png)

The Login page is the entry point to the SimpleSphere Student Portal. Students use their credentials to securely access the system.

## How to Access

- Open your web browser and navigate to the SimpleSphere LMS URL provided by your institution.
- You will be directed to the Login page automatically.

## Login Steps

**Step 1** " Enter your **Username** in the 'Username' field.

**Step 2** " Enter your **Password** in the 'Password' field. Click the eye icon to show or hide the password.

**Step 3** " _(Optional)_ Check **'Stay logged in'** to remain logged in across browser sessions.

**Step 4** " Click the **Sign In** button.

**Step 5** " Alternatively, click **Sign in with Google** to log in using your Google account.

<Note title="Forgot Password?">
If you have forgotten your password, click **'Forgot Password'** to receive a reset link to your registered email address.
</Note>

## After Login

Once logged in, students are automatically redirected to the **Student Dashboard**. The left sidebar and top header become visible with all navigation options.

## Login Page Features

| UI Element | Description |
|------------|-------------|
| **Username Field** | Input box with user icon |
| **Password Field** | Input box with lock icon and eye toggle |
| **Stay Logged In** | Session persistence checkbox |
| **Sign In Button** | Purple button with hover glow effect |
| **Sign In with Google** | OAuth SSO pill-shaped button |
| **Forgot Password?** | Recovery link for password reset |
| **Theme Toggle** | Switch between dark and light mode |

## Validation Rules

- Username field must not be empty
- Password field must not be empty
- Incorrect credentials display an error banner
- Network errors are shown in the same error alert

<Warning title="Session Security">
If you use a shared or public computer, do **not** check "Stay logged in". Always log out when you are done to protect your account.
</Warning>
