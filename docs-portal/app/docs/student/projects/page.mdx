# Projects

![Visual Guide](/images/students/project.png)

The Projects page shows all project assignments shared with you by your mentors. Projects are structured learning tasks that go beyond regular assessments.

## How to Access

- Click **'Projects'** in the left sidebar under **MY DASHBOARD**.

## Project Cards

Each project card displays:

| Field | Description |
|-------|-------------|
| **Project Title** | Name of the project |
| **Description** | Overview of the project requirements |
| **Due Date** | The deadline for project completion |
| **Status Badge** | Current project status |
| **Domain Badge** | The technology or subject area (e.g., Web Dev, Data Science) |

## Project Status Badges

| Status | Description |
|--------|-------------|
| **Assigned** | Project has been assigned, work can begin |
| **In Progress** | You have started working on the project |
| **Submitted** | You have submitted your work for review |
| **Approved** | Mentor has approved your submission |
| **Needs Revision** | Mentor has requested changes |

## Opening a Project

**Step 1** " Navigate to **Projects** in the sidebar.

**Step 2** " Click on a **project card** to open the Project Detail page.

**Step 3** " The detail page shows:
  - Full project description and requirements
  - Milestone breakdown (if configured by mentor)
  - Submission history
  - Mentor feedback and comments

## Project Detail Page

Inside a project's detail page, you can:

| Action | Description |
|--------|-------------|
| **View Instructions** | Read the full project brief and requirements |
| **Submit Work** | Upload or link your completed project |
| **View Feedback** | Read mentor comments on your submission |
| **Check Milestones** | View the project's structured milestone timeline |
| **Add Comments** | Communicate with your mentor via the comments section |

## Submitting Project Work

**Step 1** " Open the project detail page.

**Step 2** " Click the **Submit** button.

**Step 3** " Attach your work (file upload or repository link).

**Step 4** " Add a brief description of what you submitted.

**Step 5** " Click **Submit Now**.

**Step 6** " The status updates to **Submitted** and your mentor is notified.

<Info title="Project Milestones">
Some projects are broken into **milestones** " smaller deliverable stages. Complete each milestone in order and submit before the milestone deadline to stay on track.
</Info>

<Note title="Project vs Assessment">
**Assessments** are short text-based tasks. **Projects** are larger, multi-week deliverables typically involving code, design, or research output. Projects may include milestone-based reviews.
</Note>
