# Chats

![Visual Guide](/images/students/chats.png)

The Chats section allows direct messaging between students and their assigned mentors within the SimpleSphere platform.

## How to Access

- Click **'Chats'** in the left sidebar under **MY DASHBOARD**.

## Interface Overview

| Element | Description |
|---------|-------------|
| **Contact List (Left Panel)** | Lists your chat conversations with mentors |
| **Message Thread (Right Panel)** | Displays the full conversation with the selected mentor |
| **Input Box** | Type your message at the bottom |
| **Send Button** | Submits your message |
| **Timestamps** | Each message shows sender name and time |

## How to Send a Message

**Step 1** " Navigate to **Chats** in the sidebar.

**Step 2** " Select a mentor conversation from the **left panel** contact list.

**Step 3** " The message thread loads in the **right panel**.

**Step 4** " Type your message in the **input box** at the bottom.

**Step 5** " Press **Enter** or click the **Send** button.

**Step 6** " Your message appears in the thread immediately.

## Receiving Messages

- Messages from your mentor appear on the **left side** of the thread with their name and timestamp.
- New unread messages are indicated by a **notification badge** on the Chats menu item in the sidebar.

## File Sharing

You can share files with your mentor through the chat:

**Step 1** " Click the **Paperclip (Attachment)** icon in the input area.

**Step 2** " A file upload dialog opens (powered by Cloudinary).

**Step 3** " Select the file from your device.

**Step 4** " Add an optional message text.

**Step 5** " Click **Send**.

### Supported File Types for Sharing

| Type | Formats |
|------|---------|
| **Images** | JPG, PNG, GIF |
| **Documents** | PDF, DOCX, XLSX, PPTX, TXT |
| **Archives** | ZIP |
| **Media** | MP4, MP3 |

### File Size Limit

Maximum file size: **10 MB** per upload.

<Info title="Response times">
Mentors are expected to respond to messages within their SLA response window. If you don't hear back within 48 hours, use the Doubt Resolution forum or contact your admin.
</Info>

<Note title="Chat is mentor-to-student only">
Students can only message their **assigned mentors**. You cannot message other students through the chat system.
</Note>
