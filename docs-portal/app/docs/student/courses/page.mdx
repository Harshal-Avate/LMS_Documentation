# My Courses

![Visual Guide](/images/students/my-courses.png)

The Courses page is where you access all enrolled courses and explore new ones available on the platform.

## How to Access

- Click **'My Courses'** in the left sidebar.
- Or click **'All Courses'** on the Dashboard's Continue Learning section.

## Tabs

### My Courses Tab

Shows only the courses you are enrolled in. Each card displays:

| Element | Description |
|---------|-------------|
| **Thumbnail** | Course cover image (or book icon if none set) |
| **Category Badge** | Displayed top-left on the thumbnail |
| **MASTERY Badge** | Shown top-right when the course is completed |
| **Progress Bar** | Shows your completion percentage below the thumbnail |
| **Level Badge** | Beginner / Intermediate / Advanced |
| **Duration** | Estimated course length |
| **Progress %** | Numeric completion percentage |
| **Course Title & Description** | Name and brief overview of the course |
| **RESUME JOURNEY** | Button to open the course player |
| **REVIEW NODE** | Shown when course is completed " revisit content |

### Explore Tab

Shows all available courses that you have **not** yet enrolled in. Each card shows:

- Course details similar to My Courses
- **Price** (or FREE label for free courses)
- **ACQUIRE ACCESS** button " click to enrol (opens payment flow via Razorpay for paid courses)

## Search & Filters

| Feature | Description |
|---------|-------------|
| **Search Bar** | Filter courses by title or category |
| **All Levels** | Show all difficulty levels |
| **Beginner** | Filter to beginner-level courses only |
| **Intermediate** | Filter to intermediate-level courses only |
| **Advanced** | Filter to advanced-level courses only |

## Summary Stats (My Courses Tab)

Three stat cards at the top show:
- **Total Enrolled** " number of courses you are enrolled in
- **Completed** " number of courses you have finished
- **In Progress** " number of courses you are actively studying

## Resuming a Course

**Step 1** " Navigate to **My Courses** in the sidebar.

**Step 2** " Find the course you want to continue on the grid.

**Step 3** " Click **RESUME JOURNEY** to open the course player.

**Step 4** " The player will open at your last watched lesson.

## Enrolling in a New Course

**Step 1** " Click the **Explore** tab.

**Step 2** " Browse or search for a course you want to enrol in.

**Step 3** " Click **ACQUIRE ACCESS**:
  - For **free courses**: You'll be enrolled immediately.
  - For **paid courses**: A Razorpay payment dialog will open to complete the purchase.

**Step 4** " After enrollment, the course appears in your **My Courses** tab.

## Split View / Interactive Lab Mode

The Interactive Lab lets you watch video lessons and write code simultaneously:

**Step 1** " Click **'Split View Lab'** in the sidebar or the green promo card on the Dashboard.

**Step 2** " The courses page title changes to **'Interactive Lab'**.

**Step 3** " Click **'INITIALIZE LAB'** on any course.

**Step 4** " The video player and code editor appear side-by-side in a split-screen layout.

<Info title="Interactive Lab">
The Split View lab is perfect for programming courses where you need to watch a tutorial and write code at the same time without switching windows.
</Info>

<Note title="MASTERY Badge">
The **MASTERY** badge appears on a course card when you have completed all lectures, passed all quizzes with 75%+, and submitted all assignments. It also triggers certificate generation.
</Note>
