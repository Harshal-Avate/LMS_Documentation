# Certificates

![Visual Guide](/images/students/certificates.png)

The Certificates page displays all course completion certificates you have earned on the SimpleSphere platform.

## How to Access

- Click **'Certificates'** in the left sidebar under **LEARNING RESOURCES**.

## Certificate Cards

Each certificate card displays:

| Field | Description |
|-------|-------------|
| **Course Name** | The course for which the certificate was issued |
| **Verified Badge** | ✅ Confirms authenticity of the certificate |
| **Certificate ID** | Unique alphanumeric identifier for this certificate |
| **Issued Date** | When the certificate was generated |
| **Course Duration** | Length of the course (e.g., 03 MONTHS) |
| **Download PDF Button** | Downloads the certificate as a PDF document |

## Downloading a Certificate

**Step 1** " Find the certificate card for the desired course.

**Step 2** " Click **Download Certificate PDF**.

**Step 3** " The PDF is generated and automatically saved to your device's default download folder.

<Note title="Certificate Verification">
Certificates can be independently verified at **learn.simplesphere.in/certificates** using the **Certificate ID** shown on the card. Share this ID with employers or institutions to verify your achievement.
</Note>

## How to Earn a Certificate

To receive a completion certificate for a course, you must complete all of the following:

| Requirement | Minimum Threshold |
|-------------|------------------|
| **All Lectures** | Watch 100% of course lectures |
| **All Assignments** | Submit all assigned homework |
| **All Quizzes** | Score **75% or above** on each quiz |
| **Attendance** | Meet the minimum attendance requirement |

Once all conditions are met, the certificate is **issued automatically** by the system.

## Refreshing for New Certificates

If you have just completed a course and the certificate is not yet showing:

**Step 1** " Click the **Refresh** button in the top-right corner of the Certificates page.

**Step 2** " The system will reload and check for newly issued certificates.

**Step 3** " Your new certificate card will appear if the requirements have been met.

<Success title="Certificate Issued!">
When a new certificate is issued, you will also see a notification on your dashboard. The **Certificates** stat card count on your dashboard will increment automatically.
</Success>

## Certificate PDF Contents

The downloaded certificate PDF includes:
- Your full name (automatically formatted)
- Course title and duration
- Unique Certificate ID
- Course start and end dates
- Official SimpleSphere Learning branding and signature

<Warning title="Certificate ID Must Be Unique">
Each Certificate ID is unique per student. If you suspect a verification conflict or if your Certificate ID is invalid, contact your admin immediately.
</Warning>
