# My Documents

![Visual Guide](/images/students/my-documents.png)

The Documents page lets you upload and manage your personal academic documents securely on the SimpleSphere platform.

## How to Access

- Click **'My Documents'** in the left sidebar under **LEARNING RESOURCES**.

## Document Cards

Each uploaded document shows:

| Field | Description |
|-------|-------------|
| **File Icon** | Visual indicator of the file type |
| **Document Type** | The category label of the document |
| **Upload Date** | When the file was uploaded |
| **File Extension Badge** | PDF, DOCX, PNG, etc. |
| **Open Icon** | Opens the file in a new browser tab |
| **Delete Icon** | Deletes the document after a confirmation prompt |

## Supported Document Types

When uploading, you can select from the following document type categories:

| Type | Description |
|------|-------------|
| **Offer Letter** | Job or internship offer letters |
| **Internship Certificate** | Proof of internship completion |
| **Experience Letter** | Work experience confirmation |
| **Marksheet / Scorecard** | Academic score records |
| **Other** | Any other document (custom name required) |

## Uploading a New Document

**Step 1** " Click the **Upload New** button in the top-right corner.

**Step 2** " A modal appears. Select the **Document Type** from the dropdown:

**Step 3** " If you select **'Other'**, enter a custom document name in the text field that appears.

**Step 4** " Click the file area or **drag-and-drop** to select a file from your device.

**Step 5** " Click **Save Document** to upload.

**Step 6** " To cancel, click **Cancel** to close the modal without uploading.

## Opening a Document

- Click the **Open** icon (external link icon) on any document card.
- The file opens in a new browser tab via its secure Cloudinary URL.

## Deleting a Document

**Step 1** " Click the **Delete** icon (trash can) on the document card.

**Step 2** " A confirmation dialog appears asking: _"Are you sure you want to delete this document?"_

**Step 3** " Click **Confirm** to permanently delete the file.

<Warning title="Document Deletion is Permanent">
Deleted documents **cannot be recovered**. Ensure you have a local backup of important files before deleting them from the platform.
</Warning>

## File Size & Format Limits

| Constraint | Limit |
|------------|-------|
| **Maximum File Size** | 10 MB per file |
| **Accepted Formats** | PDF, DOCX, XLSX, PNG, JPG |

<Info title="Document Security">
Your documents are stored securely on Cloudinary with access controlled by your login session. Only you and your assigned admin/mentor can view your uploaded files.
</Info>
